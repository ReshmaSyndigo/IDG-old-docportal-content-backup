Type: String Indicates the locale. You can specify one of the following values:
  • _DEFAULT: If you specify the locale as _DEFAULT, then the default locale as specified in the tenant config is considered.
  • "": If you specify an empty string in the locale, then the first locale code retrieved is considered.
  • <<LocaleCode>>: If you specify a valid IETF locale format, then the locale as specified in the LocaleCode is considered.
