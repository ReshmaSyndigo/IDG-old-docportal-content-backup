This section describes some administrative and prerequisite steps you must follow before launching the Workflow Designer. **To launch the workflow designer successfully:**
  1. Open Internet Explorer browser.
  2. Click Tools > Internet Options.

    3. Result: The Internet Options window is displayed.

    {% picture prereq_wk_designer_tools_settings.png alt="Internet Options" %}
  3. Click Security > Internet, to set the Internet security settings.
    4. {% picture prereq_wk_designer_tools_settings_security_IE.png alt="Internet security settings" %}
  4. On clicking the Internet option, click Custom level.

    5. Result: The Security Setting - Internet Zone dialog box is displayed with the default settings.

    {% picture prereq_wk_designer_tools_settings_security_default.png alt="Security Setting - Internet Zone" %}
  5. Under Settings > .NET Framework, select Enable option (radio button) for Loose XAML, XAML browser applications, and XPS documents.
    6. {% picture prereq_wk_designer_tools_settings_security_IE_XAML.png alt="XAML Settings" %}
  6. Under Settings > Downloads, select Enable option (radio button) for File download, Font download, and Enable .NET Framework setup.
    7. {% picture prereq_wk_designer_tools_settings_security_IE_download.png alt="IE Download" %}
  7. Click OK to save internet security settings.
  8. Click Security > Trusted Sites, to set the security settings for trusted sites
    9. {% picture prereq_wk_designer_tools_settings_security.png alt="Trusted Sites Security" %}
  9. On clicking the Trusted sites, click Custom level.

    10. Result: The Security Setting - Trusted Sites Zone dialog box is displayed with the default settings.

    {% picture wk-trusted-sites.png alt="Security Setting - Trusted Sites Zone" %}
  10. Under Settings > .NET Framework, select Enable option (radio button) for Loose XAML, XAML browser applications, and XPS documents
    11. {% picture wk-trusted-sites-dialog.png alt="XAML Setting" %}
  11. Under Settings > Downloads, select Enable option (radio button) for File download, Font download, and Enable .NET Framework sites.
    12. {% picture wk-trusted-sites-dialog-box.png alt="Trusted Sites Security" %}
  12. Click OK to save security settings on trusted sites.
  13. Click Security > Trusted Sites, to set the trusted sites.
    14. {% picture prereq_wk_designer_tools_settings_security_trustedsites.png alt="Trusted Sites Security Setting" %}
  14. Click Sites

    15. Result: The Trusted sites dialog box is displayed with default settings.

    {% picture prereq_wk_designer_tools_settings_security_trustedsites_dialog.png alt="Trusted Sites" %}
  15. To include current URL of the MDM application to the zone, enter the website address (URL) (example https://mahiweb12), and then click Add.

    16. Result: The newly added website URL is displayed in Websites list.

  16. Click Close to close the Trusted sites dialog box.
  17. Click OK on exit the Internet Options window.
