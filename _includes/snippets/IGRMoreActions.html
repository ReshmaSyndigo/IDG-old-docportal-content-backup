More Actions on Selected Entities

    You can further perform these actions on selected entities with the help of following options provided:
  • Bulk Edit Selected Entities: Provides an option to bulk edit selected entities report list.
  • Open Entities in Entity Explorer: Provides an option to open the selected entities in the Entity Explorer page. In Entity Explorer, you can perform necessary actions on the selected entities. To know more about actions that can be performed on entities from the Entity Explorer page, see Perform Basic Actions on Entities

If you wish to select entities in all the pages, you can click the check box in the header row and choose "All pages" option.

{% picture jig-more-actions.png alt="Select Entities" %}

To open entities in the Entity Explorer page, first select the required entities from the table and then click on Entity Explorer icon.

Note: To open the entities in Entity Explorer, you must filter out the entities of entity type 'Category' manually. Only the entities with the entity type other than 'Category' are displayed in the Entity Explorer page.

{% picture mm-open-select-entities-ee.png alt="Open Select Entities" %}

Result: The selected entities are displayed in the Search Results grid of the Entity Explorer page.

{% picture jig-more-action-result.png alt="Entities in Entity Explorer" %}
